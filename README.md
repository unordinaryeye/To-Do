# Daily Routine 사용 가이드

나만의 루틴 체크리스트 앱입니다. 아이폰 홈 화면에 추가하면 네이티브 앱처럼 사용할 수 있어요.

---

## 1. 배포하기 (GitHub Pages)

1. [github.com](https://github.com) 로그인
2. 우상단 **+** → **New repository** 클릭
3. 이름 입력 (예: `routine`), **Public** 선택 → **Create repository**
4. **"uploading an existing file"** 링크 클릭
5. `routine-checklist-pwa.html` 파일 이름을 **`index.html`**로 변경 후 드래그 앤 드롭
6. 하단 **Commit changes** 클릭
7. 레포 상단 **Settings** → 왼쪽 **Pages**
8. Source: **Deploy from a branch** → Branch: **main** / **/ (root)** → **Save**
9. 1~2분 후 `https://유저이름.github.io/routine/` 으로 접속 가능

---

## 2. 아이폰에 설치하기

1. **Safari**로 배포된 URL에 접속
2. 하단 **공유 버튼** (↑) 탭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **추가** 탭
5. 홈 화면에 앱 아이콘이 생기면 완료

> 반드시 Safari에서 진행해야 합니다. Chrome에서는 홈 화면 추가가 지원되지 않아요.

---

## 3. 기능 사용법

### 체크리스트 탭

- 오늘의 루틴 목록이 카테고리별로 표시됩니다
- 항목을 **탭하면 체크/해제**
- 상단 **진행률 바**에서 오늘의 달성률 확인
- **연속 달성일**(스트릭)도 자동 계산됩니다
- 좌우 **화살표(‹ ›)**로 날짜 이동, 지난 기록도 확인 가능
- **주간 캘린더**의 점 색상으로 한눈에 상태 파악
  - 🟢 초록: 전체 완료
  - 🟡 노랑: 일부 완료
  - ⚪ 회색: 미완료

### 루틴 관리 탭

- **이모지 선택** → 루틴 이름 입력 → 카테고리 선택 → **추가** 버튼
- 기존 루틴의 **이름을 탭**하면 수정 모드 (입력 후 Enter 또는 다른 곳 탭)
- **× 버튼**으로 루틴 삭제

### 카테고리 종류

| 카테고리 | 용도 |
|---------|------|
| 🌅 모닝 루틴 | 아침에 하는 습관 |
| 💼 업무 | 업무 관련 루틴 |
| 💪 건강 | 운동, 건강 관리 |
| 🌙 이브닝 루틴 | 저녁/취침 전 습관 |
| 🧘 셀프케어 | 명상, 자기관리 |
| 📚 학습 | 공부, 독서 |

## 4. 데이터 백업 & 복원

데이터는 기기 내(localStorage)에 저장됩니다. 앱을 오래 안 열면 iOS가 데이터를 삭제할 수 있으니, 주기적으로 백업하세요.

### 백업하기

1. 우상단 **⚙️ 설정** 버튼 탭
2. **💾 데이터 백업** 탭
3. JSON 파일이 "파일" 앱에 저장됨
4. 마지막 백업 날짜가 자동으로 기록됩니다

### 복원하기

1. **⚙️ 설정** → **📂 데이터 복원** 탭
2. 파일 선택 화면에서 백업 JSON 파일 선택
3. 즉시 복원 완료

### 백업 리마인더

- 7일 이상 백업하지 않으면 체크리스트 상단에 **노란 배너**가 표시됩니다
- 배너를 탭하면 설정 화면으로 바로 이동합니다

### 초기화

- **⚙️ 설정** → **🗑️ 전체 초기화**
- 확인 팝업에서 "초기화" 탭하면 모든 데이터가 삭제되고 기본 루틴으로 돌아갑니다

## 6. 참고사항

- **Safari 전용**: 홈 화면 앱(PWA)은 Safari에서만 추가 가능합니다
- **오프라인 사용**: 한 번 로드된 후에는 인터넷 없이도 사용 가능합니다
- **데이터 위치**: 모든 데이터는 기기 내 localStorage에 저장되며, 외부 서버로 전송되지 않습니다
- **백업 권장 주기**: 일주일에 한 번 이상
